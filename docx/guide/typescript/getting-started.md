
# 快速开始

## 环境

- 安装 [Visual Studio Code](https://code.visualstudio.com/)
- 安装Visual Studio Code的插件`TypeScript`、`Code Runner`
- 安装编译环境

```bash
npm install -g typescript
```

编译`ts`文件

```bash
tsc index.ts
```
- 安装运行环境

```bash
npm install -g ts-node tslib @types/node
```
运行`ts`文件
```bash
ts-node index.ts
```
