# Git History

Git History 是一个可以轻松快速浏览 GitHub 文件操作历史记录的工具，该工具的功能和使用方法简洁明了，只要将任何文件的 URL 中的“github.com”替换成“github.githistory.xyz”即可以动画的方式快速查看该文件的修改历史记录。 查看详细信息 >>open in new window
