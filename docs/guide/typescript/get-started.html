# 环境

  • 安装 Visual Studio Codeopen in new window
  • 安装Visual Studio Code的插件TypeScript、Code Runner
  • 安装编译环境
npm install -g typescript

编译ts文件

tsc index.ts
  • 安装运行环境
npm install -g ts-node tslib @types/node

运行ts文件

ts-node index.ts
